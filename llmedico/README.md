### Translate

minimum:

llmedico translate --target org.project.MyClass \
--src /data/input/project/src \
--jar /data/input/project/target/project.jar \
--out /data/output

| Option                       | Description                                                                                                                                                            |
|------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `--target`, `--target-class` * | Fully-qualified name of the class for which LLMedico has to generate conditions. _e.g. org.project.MyClass_                                                            |
| `--target-method`            | Method of class for which LLMedico has to generate test oracles. _e.g. isNotEmpty_                                                                                     |
| `--src`, `--src-dir` *       | Directory containing all the source files of the project. _e.g. /data/input/project/src/_                                                                              |
| `--jar`, `--jar-dir` *       | The project compiled as a jar. _e.g. /data/input/project/target/project.jar_                                                                                           |
| `--out-dir`                  | Directory to save every file generated by LLMedico. _e.g. /data/output/_  If need to specify the files to be outputted, else they will be printed to the command line. |
| `--log`                      |                                                                                                                                                                        |
| `--extractions`              |                                                                                                                                                                        |
| `--translations`             | _for debugging purposes_                                                                                                                                               |
| `--conditions-llmedico`      |                                                                                                                                                                        |
| `--conditions-toradocu`      |                                                                                                                                                                        |
| `--help`, `-h`               | Print the list of available options.                                                                                                                                   |
| `--debug`                    | Enable fine-grained logging.                                                                                                                                           |
| `--silent`                   | Disable all output to the command line.                                                                                                                                |


### Evaluate
minimum:
llmedico evaluate --expected goal-conditions-org.project.MyClass.json \
--generated llmedico-conditions-org.project.MyClass.json \
--result result-org.project.MyClass.csv

| Option              | Description                                                                                        |
|---------------------|----------------------------------------------------------------------------------------------------|
| `--expected`, *     | Path to the expected-conditions. _e.g. /data/output/goal-conditions-org.project.MyClass.json_      |
| `--expected-type`, * | Specify the format. _{llmedico, jdoctor}_                                                          |
| `--generated`, *    | Path to the generated-conditions. _e.g. /data/output/llmedico-conditions-org.project.MyClass.json_ |
| `--generated-type`, * | Specify the format. _{llmedico, jdoctor}_                                                          |
| `--out`, *          | Path to save the results. _e.g. /data/output/_                                                     |
| `--result`          | Path to save the results. _e.g. /data/output/llmedico-evaluation-org.project.MyClass.csv_          |
| `--debug`           | Enable fine-grained logging.                                                                       |
| `--silent`          | Disable all output to the command line.                                                            |
| `--help`, `-h`      | Print the list of available options.                                                               |
